This repository is a frontend assignment from Codeliner created to evaluate frontend development skills, code quality, problem-solving ability, and attention to detail.

---

## 🚀 Technology Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS 4
- Framer Motion
- Lucide React Icons
- ESLint

## ⚙️ Local Setup

### 1. Clone the repository

```bash
git clone <repository-url>
cd n7-app
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```

### 4. Open the application

Visit the local URL shown by Vite in your browser:

```text
http://localhost:5173
```
